NAME

ins_ts_heya_tinryo_hist_start.sql

DESCRIPTION

賃料履歴マスタ(TS_HEYA_TINRYO_HIS)のTIN_HIS_YMD（履歴カラムに） END_FLG!=1(提供継続またはSTART_FLG=1）情報よりも END_FLG=1（提供が終了）している情報のほうが新しい場合

今回のデータに該当建物(TATEMONO_CD)＋部屋番号（ROOMNO)の情報が含まれれば データ提供が再スタートされたとみなして、 START_FLG＝１のレコードをインサートします。

TIN_HIS_YMDには今回のデータ提供日が入ります

Trouble Shooting

TOUROKUYMD が最新でSTART_FLG=1のレコードを消すオペレーションが必要

AUTHOR

システム部　瀧内誠(mtakiuc@kantei.co.jp)
