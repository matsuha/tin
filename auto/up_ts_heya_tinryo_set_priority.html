NAME

up_ts_heya_tinryo_set_priority.sql

DESCRIPTION

賃料マスタ(TS_HEYA_TINRYO)の最優先情報をこの段階で更新する

まず、TS_HEYA_TINRYOのPRIORITY=0にして初期化する。 最優先候補１番PRIORITY_SELECT=1のレコードをPRIORITY=1 に更新する。 同情報を建物マスタ（TS_TATEMONO にも反映する。 TS_TATEMONO.PRIORITY=1に設定する。 (TS_TATEMNO.PRIORITYも事前に０に初期化する） =head1 HISTORY
