NAME

ins_ts_heya_tinryo_hist_end.sql

DESCRIPTION

賃料履歴マスタ(TS_HEYA_TINRYO_HIS)の情報をインサートする

前回データ提供がされた建物(TATEMONO_CD)＋部屋（ROOMNO) の情報が、今回のデータに含まれていない場合 END_FLG=1の情報を該当建物CD＋ROOMNOでインサートする。

TIN_HIS_YMD(今回のデータ供給日）
END_YMD(前回のデータ提供日）
END_YMD_SUM(前回のデータ提供日) 同一空室期間内で複数レコードが存在する際 START_YMD_SUM,END_YMD_SUMで空室期間全体を表す

HISTORY

  1. /10/25 M.TAKIUCHI CREATED
  2. /10/25 M.TAKIUCHI COMMENT ADD FOR POD

Trouble Shooting

最新TOUROKUYMDで入力されたEND_FLG=1の情報を削除して、再設定する
